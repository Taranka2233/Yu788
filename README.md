# Найт-Сити Нет — Cyberpunk мессенджер

Мессенджер в стиле Cyberpunk 2077. Собирается в Android APK автоматически через GitHub Actions.

## Как получить APK

1. Залей все эти файлы в новый репозиторий GitHub (сохраняя структуру папок, включая `.github/workflows/`).
2. Открой вкладку **Actions** — сборка запустится сама после первого коммита.
3. Дождись зелёной галочки (~5–10 минут), открой завершённый запуск.
4. В разделе **Artifacts** скачай **app-debug-apk** — это готовый APK.
5. Перекинь APK на телефон и установи (разреши установку из неизвестных источников).

## Структура

- `index.html` — само приложение (одним файлом, всё внутри).
- `.github/workflows/build-apk.yml` — конвейер сборки APK (Capacitor + Gradle).
- `capacitor.config.json`, `package.json` — конфигурация Capacitor и плагины.
- `setup_icons.py` — добавляет динамические иконки приложения (как в Telegram).
- `appicon_01..10.png`, `icon.png` — иконки.

## Обновить приложение

Замени `index.html` на новую версию и сделай commit — APK пересоберётся автоматически.
